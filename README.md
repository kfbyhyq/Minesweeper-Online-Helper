# Minesweeper Online Helper 在线扫雷网（WoM）助手
https://minesweeper.online/
## v1.0
三个按钮：

1. 刷新宝石/场币价格：

   作用页面：商店主页

   功能：提取页面宝石、竞技场币、装备碎片当前最低市场价

   输出：csv文件
   
3. 刷新竞技场门票价格：

   作用页面：竞技场主页

   功能：提取所有竞技场门票（必须是当前账号拥有的）当前最低市场价

   输出：csv文件
   
5. 刷新个人数据：

   作用页面：我的主页（需要中文）

   功能：提取账号现有资源

   输出：csv文件
